Manipulando datas e horas

Para se obter os dados separadamente, existem os seguintes métodos:

  • from datetime import date
  • DataToda=date.today()
  • DataToda.day - Retorna o dia do mês (inteiro entre 1 e 31)
  • DataToda.weekday() - Retorna o dia da semana (0=Segunda-feira,... , 5=Sábado, 6=Domingo)
  • DataToda.month - Retorna o mês (entre 1=Janeiro e 12=Dezembro)
  • DataToda.year - Retorna o ano com quatro dígitos (o mais usado)
  • from datetime import datetime
  • now = datetime.now()
  • now.strftime("%H") - Retorna a hora do dia (entre 0 e 23)
  • now.strftime("%M") - Retorna os minutos dentro da hora (entre 0 e 59)
  • now.strftime("%S") - Retorna os segundos (entre 0 e 59)

No exemplo 1 abaixo obteremos o dia da semana e as horas. Para tal, utilizaremos a variável DataToda para armazenar data e hora.

Exemplo 1

from datetime import date
from datetime import datetime

DataToda=date.today()

Dia_do_mes=DataToda.day
Dia_da_semana=DataToda.weekday()
Mes=DataToda.month
Ano=DataToda.year
# ---------------------------------------
now = datetime.now()
Horas=now.strftime("%H")
Minutos=now.strftime("%M")
Segundos=now.strftime("%S")
# ---------------------------------------
semana = ("Segunda", "Terça", "Quarta", "Quinta", "Sexta", "Sabado", "Domingo")
Semana_atual=semana[Dia_da_semana]
#-------------------------
Mes2=("", "Janeiro", "Fevereiro", "Março", "Abril", "Maio", "Junho", "Julho", "Agosto", "Setembro", "Outubro", "Novembro", "Desembro")
Mes_atual=Mes2[Mes]
print(f"{Semana_atual}, {Dia_do_mes} de {Mes_atual} de {Ano}")
print(f"São: {Horas} horas, {Minutos} minutos e {Segundos} segundos.")
